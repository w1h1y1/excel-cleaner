# Excel Data Cleaner Tool

A simple Python tool to automatically clean messy Excel data.

## 🚀 What This Tool Can Do
- Remove duplicate rows
- Remove rows with missing values
- Sort data by any column (e.g., salary)
- Export a clean and organized Excel file

## 💡 Use Cases
This tool is useful for:
- Cleaning customer data
- Preparing business reports
- Removing duplicate entries from large datasets
- Organizing messy Excel files

## 🛠 Tech Stack
- Python
- pandas
- openpyxl

## ▶️ How to Run
1. Place your Excel file in the same folder as `main.py`
2. Rename it to `sample.xlsx`
3. Run:
```bash
python main.py
